## 📌 项目概述

**NarratorAI** 是一个强大的多媒体 AI 智能处理平台，专为内容创作者、短剧制作团队、MCN 机构和出海企业打造。我们为用户提供一站式视频处理解决方案，包括字幕提取、翻译、擦除、压制以及 AI 解说等核心功能，助力您的内容快速实现全球化和本地化。

> **核心优势：** 业界领先的 AI 引擎提供无与伦比的处理质量和效率，降低 90% 人工成本，提升 10 倍内容生产速度。

## 💪 我们的核心能力

### 1️⃣ 专业视频翻译

- **全流程自动化：** 上传一个视频，API 会自动完成提取字幕、翻译内容、生成字幕、压制合成的完整流程
- **字幕无痕擦除：** 智能识别并无痕去除视频中的原有硬字幕，保持画面自然流畅
- **高保真翻译：** 采用专业领域训练的神经网络模型，确保专业术语准确性和语言自然度
- **多语言支持：** 支持所有主流语种，包括小语种等

### 2️⃣ AI 视频解说

- **智能内容理解：** 基于深度学习的视频内容分析，准确理解视频场景、动作和主题
- **专业解说生成：** 根据视频内容自动生成专业、流畅的解说词，符合视频风格和主题
- **行业适应性：** 可根据不同行业和领域调整解说风格

### 3️⃣ 字幕翻译

- **高效处理：** 直接上传 SRT 格式字幕文件，快速获得翻译结果
- **格式保留：** 完美保留原始字幕的时间轴和格式信息
- **术语一致性：** 支持自定义术语表，确保专业术语翻译准确和统一
- **翻译一致性：** 智能维持上下文语境一致性，避免片段式翻译的不连贯问题

### 4️⃣ 字幕提取

- **OCR 技术提取：** 使用高精度 OCR 技术从视频画面中识别并提取硬字幕，正确率高达 98%
- **智能处理：** 自动处理各种字体、颜色和背景的字幕显示
- **保留时间轴：** 精确定位每条字幕的出现和消失时间
- **语言支持：** 能够识别中文、英语语言

### 5️⃣ 字幕视频无痕擦除

- **智能识别技术：** 采用最新的 AI 视觉识别模型自动定位视频中的硬字幕位置
- **场景感知擦除：** 根据视频画面背景智能重建被字幕覆盖的区域
- **保持画面一致性：** 确保擦除后的画面与原始背景无缝衔接，无可察觉的修改痕迹

### 6️⃣ 视频字幕压制

- **专业压制引擎：** 使用广播级视频处理技术将 SRT 字幕文件与视频完美结合
- **字幕样式定制：** 支持自定义字体、大小、颜色、位置、边框等多种视觉效果
- **保持视频质量：** 压制过程不损失原始视频质量，支持高清、4K 等各种分辨率
- **批量处理：** 可同时处理多个视频的字幕压制任务，提高工作效率

## ⚡ 快速开始

### 环境要求

- Node.js 18.0+
- npm 8.0+

### 开始开发

1. **获取代码:**

   ```bash
   git clone https://github.com/Narrator-AI/NarratorAI.git
   cd NarratorAI
   ```

2. **安装依赖:**

   ```bash
   npm install
   ```

3. **启动演示界面:**

   ```bash
   npm run dev
   ```

### 界面操作流程

1. 启动后在界面中配置您的 API 密钥
2. 创建项目文件夹或选择现有项目文件夹
3. 上传视频或 SRT 文件
4. 选择处理类型和目标语言
5. 提交任务并查看处理进度
6. 下载或在线预览处理结果

## 📹 实际案例展示

### 字幕擦除+翻译+重新压制一体化处理

**客户挑战：** 某网络剧制作公司拥有一部 60 集的热门的中文网剧，原片中包含中文硬字幕，需要制作英语、泰语、日语和西班牙语四种语言版本，以拓展海外市场。传统流程需要多个专业团队协作，耗时 3 个月，成本高昂。

**一体化解决方案：**
1. **字幕擦除：** 使用 NarratorAI 的无痕字幕擦除 API，智能识别并清除所有原始中文硬字幕
2. **字幕提取与翻译：** 同时使用字幕提取 API 获取原始中文台词，并使用专业翻译 API 转换为目标语言
3. **翻译优化：** AI 自动调整文化表达方式，使翻译内容符合目标市场习惯
4. **字幕压制：** 将翻译后的多语言字幕自动压制回无字幕的视频中，保持专业品质
